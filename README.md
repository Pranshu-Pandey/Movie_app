# Flutter Movie Explorer

Flutter Movie Explorer is a mobile application that leverages the TMDb (The Movie Database) API to provide users with information about trending, top-rated, and upcoming movies. The app is built using Flutter, allowing a seamless cross-platform experience.

## Features

- **Trending Movies:** Explore a list of currently trending movies.
- **Top Rated Movies:** Discover a curated selection of top-rated movies.
- **Upcoming Movies:** Stay informed about upcoming movie releases.

## Getting Started

### Prerequisites

Before you start, ensure you have the following:

- Flutter SDK: Follow the [official Flutter installation guide](https://flutter.dev/docs/get-started/install) to set up Flutter on your machine.
- TMDb API Key: Obtain your API key by creating an account on the [TMDb website](https://www.themoviedb.org/) and following their instructions to generate an API key.
